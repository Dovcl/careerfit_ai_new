# CareerFit AI

> 취업·공모전 데이터 기반 맞춤형 AI 포트폴리오 코치



## 프로젝트 개요

CareerFit AI는 대학생과 취업 준비생이 자신의 전공, 기술 스택, 관심 연구 분야를 기반으로 맞춤형 공모전과 연구실을 추천받을 수 있는 AI 기반 커리어 탐색 플랫폼이다. 사용자는 자신의 역량을 입력하면 AI가 관련 데이터를 분석하여 적합도가 높은 공모전과 연구실을 추천하고, 추천 근거와 데이터 출처를 함께 제공하여 결과의 신뢰성을 높인다. 이를 통해 흩어져 있는 정보를 일일이 탐색하는 부담을 줄이고, 자신의 진로와 역량에 맞는 성장 기회를 효율적으로 발견할 수 있도록 지원하는 것을 목표로 한다.



## 기술 스택



| 영역 | 기술 |

|---|---|

| 백엔드 | Python, FastAPI |

| AI API | Gemini 2.5 Flash-Lite |

| 데이터 | Pandas, SQLite, ChromaDB |

| 프론트엔드 | React, Vite |

| 실행 환경 | Docker |
